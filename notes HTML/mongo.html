MongoDB Notes

mongoDB QUERY SELECTORS=>
Update Operators:
$currentDate : Sets the value of a field to current date
$inc : Incrementsthe value of the field by specified amount
$min : update field if the value is less than existing field value
$max : update field if the value is more than existing field value
$mul : multiplies the value of the field by the specified amount
$rename : renames a field
$set : sets the value of a field in a doc
$setOnInsert :Sets the value ofa field if an update results in an insert of a doc.
$unset : removes the specified field from a doc Comparison:
$eq : equal to
$gt : greater than
$gte : greater than or equal to
$in : any value specified in array
$lt : less than
$lte : less than or equal to
$ne : not equal to
$nin : values not in array Logical:
$and : joins query clauses with logical AND
$not : inverts the effect of query expression
$nor : logical NOR returns all docs that fail to match both clauses
$or : logical OR returns all docs those match either of the queries

Misc:
$regex : selects doc where values match a specified regex
$text : performs text search
$where : matches doc that satisfy a JS expression
$all : matches arrays that contain all elements in the query
$ : projects the first element in array that matches the query condition
$slice : limits the number of elements projected from array. Supports skip & limit slices.
$comment: adds a comment to a query predicate
$rand : generates a random float between 0 and 1
